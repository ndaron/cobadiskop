WP Login Screen is best option to change Login Screen in just one click and choose ready to use login theme.

== Description ==
Simple WP Login is very light plugin to change branding of your site. You don't need to waste time to design WP Login screen.
Just browse our screen page and select any template (we will be updating templates in future).
<h3>Features</h3>
* 12 Ready to use Login themes
* Upload Logo
* Use Text instead Logo

<h4>Login Screen Styles</h4>
* Bars Login
* Animated Login
* Batman Login
* Black Login
* Blue Login
* Sky Blue Login
* Blurred Login
* Dark Login
* Flower Login
* Silver Floor Login
* Sky Login
* Wood Skin Login

[Recommended with Our Simple Member Registration Plugin](https://wordpress.org/plugins/wp-registration/)

== Installation ==
* Upload plugin directory to the /wp-content/plugins/ directory
* Activate the plugin through the \'Plugins\' menu in WordPress
* After activation, you can set options from Settings -> WP Login
